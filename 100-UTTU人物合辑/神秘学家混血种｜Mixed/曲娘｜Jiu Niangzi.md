## 塑造

| 塑造等级 |                           塑造效果                           |
| :------: | :----------------------------------------------------------: |
|   Lv.1   | 【酒与之外的事故】造成的现实创伤提升至**400%**；每少1人，造成的额外现实创伤提升至**150%**，该效果最多提升**300%**现实创伤 |
|   Lv.2   | 【更尽—杯】每消耗1层**[酒浆]**，造成的现实创伤提升至**90%**  |
|   Lv.3   |    【存想】随机赋予自身**[增益集合]**的效果数量提升至2种     |
|   Lv.4   | 【更尽—杯】消耗**[酒浆]**的最大层数提升至4层，消耗1/2/3/4层**[酒浆]**的概率变为**5%/15%/40%/40%** |
|   Lv.5   | 【存想】**[增益集合]**中的效果提升至**25%**；回合结束时，改为每拥有3种不同的对应状态，赋予自身1层**[酒浆]**，获取上限提升至5层 |
