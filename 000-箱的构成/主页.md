## 以影像之
```dataview
Table without id
	embed(link(cover)) AS "海报",
	"<span style='display:flex; justify-content: center;'>" + "《" + "**" + Chapter+ "**" + "》" + "</span>",
	"<span style='display:flex; justify-content: center;'>" + "*" + English + "*"  + "</span>",
	"<span style='display:flex; justify-content: center;'>" + Era  + "</span>" AS "年代"
Sort Serial
Where contains(file.path,"300-以影像之") AND !contains(file.name,"300-以影像之")
```
